# Pomodoro

A simple iOS/iPadOS Pomodoro timer app that helps you focus by managing study and rest intervals.  
The app supports live notifications to keep you informed even when it's running in the background.

---

## Features

- Set custom **study** and **rest** intervals  
- Define total session duration  
- Receive **live notifications** for session updates  
- Plays **distinct sounds** for study and rest transitions  
- Designed for **iOS and iPadOS**

---

## Requirements

- iOS/iPadOS 18.1 or later  
- Xcode 16.1 or later  
- SwiftUI and Swift 5

---

## How to Run

1. Clone the repository
2. Open the project in Xcode
3. Add required sound files (not included in the repository for legal reasons):
   - Add bell.mp3 → placed in the Resources directory
     - Used when study time starts.
   - Add doneit.mp3 → placed in the Resources directory
     - Used when rest time starts.

## Notes

- Notification permissions must be granted for the app to display live updates.
- Sound files are required for proper functionality but are not distributed with this project.
